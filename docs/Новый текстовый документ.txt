В C1.6_Background_as_Contextual_and_Load_Factor.md.:
- "исторически" - фон всех стимулов всех тестов был одинаков (константа). Учет его влияния возможен с точки зрения "гипотетического" взаимодействия фона и какого-либо из тестовых цветов (возможно, есть еще какие-либо мысли на эту тему);
- в перспективных проектах - фон должен будет играть существенную роль в создании когнитивной нагрузки в ходе непрерывного дву-фазного тестирования Ф1-Ф2: фон должен соответствовать: канальной направленности конкретного теста (разный для магно-, парво-, конио-стимулов); фон должен создавать различие в когнитивной нагрузке для кокретного канала (по контрасту, цвету-оттенку или обманному движению, предшествующему тестовому сигналу). Возможно, какие-то характеристики, необходимые для корректного проектирования фона. Подскажи, если заметишь.
